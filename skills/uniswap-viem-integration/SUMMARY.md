# uniswap-viem-integration
Integrate EVM blockchains into TypeScript and JavaScript applications using viem and wagmi with wallet connection, contract reads, and transaction signing.

## Highlights
- EVM blockchain integration using the viem library for TypeScript/JavaScript
- wagmi integration for React and Next.js applications
- Wallet connection, account management, and network switching
- Contract reads and writes with full type safety via ABIs
- Transaction signing and submission workflows
- Part of the uniswap-viem plugin — prerequisite for uniswap-swap-integration
- Covers public client, wallet client, and test client setup
- Multi-chain configuration patterns for all Uniswap-supported networks
