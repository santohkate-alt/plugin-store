# uniswap-viem-integration

Integrate EVM blockchains using viem and wagmi for TypeScript and JavaScript applications

## Source

This skill is maintained by Uniswap Labs in the [uniswap-ai](https://github.com/uniswap/uniswap-ai) monorepo.

The canonical source is at [`packages/plugins/uniswap-viem/skills/viem-integration/`](https://github.com/uniswap/uniswap-ai/tree/main/packages/plugins/uniswap-viem/skills/viem-integration).

### What It Does

- Guides AI agents through setting up viem clients, accounts, and chain configurations
- Covers reading blockchain data, sending transactions, and smart contract interactions
- Provides patterns for wallet integration using wagmi
