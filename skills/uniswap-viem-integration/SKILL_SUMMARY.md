# uniswap-viem-integration — Skill Summary

## Overview
The uniswap-viem-integration skill provides comprehensive guidance for integrating EVM blockchains into TypeScript and JavaScript applications using viem and wagmi. It covers client setup (public, wallet, and test clients), account management, multi-chain configuration, typed contract reads and writes using ABIs, wallet connection flows, and transaction signing and submission. This skill is a prerequisite for the uniswap-swap-integration skill, which relies on viem fundamentals for executing swaps on-chain.

## Usage
Install via `claude plugin add @uniswap/uniswap-viem`. Use this skill when starting any TypeScript/JavaScript project that interacts with EVM chains — set up viem clients and wagmi config first, then layer Uniswap-specific integrations on top.

## Commands
This is a reference skill with no CLI commands. Integration is code-based using the `viem` and `wagmi` npm packages.

| Package | Install |
|---|---|
| viem | `npm install viem` |
| wagmi | `npm install wagmi` |

## Triggers
Activates when the user mentions viem, wagmi, EVM TypeScript integration, wallet connection in React, contract reads/writes, or setting up a blockchain client for a JavaScript or TypeScript application.
