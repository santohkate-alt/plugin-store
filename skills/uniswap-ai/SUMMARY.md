# uniswap-ai
AI-powered Uniswap developer tools covering trading, hooks, CCA auctions, swap/liquidity planning, and EVM integration across V2/V3/V4.

## Highlights
- Monorepo of 5 specialized Uniswap AI plugins (trading, hooks, CCA, driver, viem)
- Trading API integration with Universal Router SDK and direct contract call support
- Uniswap V2, V3, and V4 full coverage
- Continuous Clearing Auction (CCA) deploy and configure workflows
- Swap planner and liquidity planner with deep link generation
- V4 hooks security guide with vulnerability patterns and audit checklist
- EVM integration via viem/wagmi for TypeScript and JavaScript
- Multi-chain support across 12 chains (Ethereum, Base, Arbitrum, Optimism, Polygon, and more)
