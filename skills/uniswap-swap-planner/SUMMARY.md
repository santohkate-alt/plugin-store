# uniswap-swap-planner
Plan token swaps and generate Uniswap deep links across all supported chains, with token discovery and research workflows.

## Highlights
- Plan token swaps across all Uniswap-supported chains
- Generate Uniswap deep links for direct swap execution in the app
- Token discovery and research workflows (symbol lookup, contract resolution)
- Multi-chain support covering 12 networks
- Part of the uniswap-driver plugin alongside liquidity planner
- Works with Uniswap V2, V3, and V4 pools
- AI-guided swap planning with routing recommendations
- Pairs with uniswap-swap-integration for full programmatic swap flows
