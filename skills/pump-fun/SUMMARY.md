# pump-fun

Buy and sell tokens on pump.fun bonding curves on Solana mainnet.

**Category**: defi-protocol  
**Author**: skylavis-sky  
**License**: MIT

## Commands

- `get-price` — current price and bonding curve state
- `get-token-info` — token metadata and creator info
- `buy` — buy tokens with SOL
- `sell` — sell tokens for SOL
