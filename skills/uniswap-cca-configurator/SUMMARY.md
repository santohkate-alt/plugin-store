# uniswap-cca-configurator
Configure Continuous Clearing Auction (CCA) smart contract parameters for fair, transparent, and manipulation-resistant token distribution.

## Highlights
- Configure all CCA smart contract parameters for token distribution auctions
- Fair and transparent token distribution mechanism on Uniswap V4
- Covers auction duration, starting price, clearing price, and allocation parameters
- Part of the uniswap-cca plugin suite — pairs with uniswap-cca-deployer
- Targets Ethereum and EVM-compatible chains
- AI-guided parameter selection with validation and sanity checks
- Designed to prevent common CCA misconfiguration vulnerabilities
- References official Uniswap CCA documentation and example configurations
