# uniswap-cca-configurator

Configure Continuous Clearing Auction (CCA) smart contract parameters for fair and transparent token distribution

## Source

This skill is maintained by Uniswap Labs in the [uniswap-ai](https://github.com/uniswap/uniswap-ai) monorepo.

The canonical source is at [`packages/plugins/uniswap-cca/skills/configurator/`](https://github.com/uniswap/uniswap-ai/tree/main/packages/plugins/uniswap-cca/skills/configurator).

### What It Does

- Guides AI agents through configuring CCA auction parameters
- Covers supply schedules, pricing curves, and auction timing
- Generates JSON configuration files ready for deployment

### Related Skills

- **uniswap-cca-deployer**: Deploy configured CCA contracts on-chain
