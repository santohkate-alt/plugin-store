# uniswap-liquidity-planner

Plan and generate deep links for creating liquidity positions on Uniswap v2, v3, and v4

## Source

This skill is maintained by Uniswap Labs in the [uniswap-ai](https://github.com/uniswap/uniswap-ai) monorepo.

The canonical source is at [`packages/plugins/uniswap-driver/skills/liquidity-planner/`](https://github.com/uniswap/uniswap-ai/tree/main/packages/plugins/uniswap-driver/skills/liquidity-planner).

### What It Does

- Plans concentrated liquidity positions with price range selection
- Generates deep links to create positions in the Uniswap web interface
- Supports Uniswap v2, v3, and v4 pool types

### Related Skills

- **uniswap-swap-planner**: Plan token swaps alongside liquidity provision
- **uniswap-viem-integration**: Foundational EVM blockchain integration
