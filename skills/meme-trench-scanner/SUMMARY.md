# meme-trench-scanner
Fully automated Solana meme token trading bot with 11 launchpad coverage, triple signal detection, 7-layer exit system, and TraderSoul AI observation — driven by onchainos CLI with TEE signing.

## Highlights
- 11 Solana launchpad coverage (pump.fun, Believe, LetsBonk, and more)
- Triple signal detection: TX acceleration + volume surge + buy/sell ratio
- 5m/15m precision buy/sell ratio computed from raw trade data
- Deep safety checks: dev rug history, bundler holdings, LP lock, aped wallets
- 7-layer exit system: emergency exit, FAST_DUMP crash detection, stop loss, trailing stop, tiered TP
- TOP_ZONE price position filter (skips tokens near ATH >85%)
- TraderSoul AI observation system with personality tags
- Web dashboard at localhost:3241; Paper Mode default, Live Mode requires explicit confirmation
