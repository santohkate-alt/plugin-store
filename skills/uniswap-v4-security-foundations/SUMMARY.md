# uniswap-v4-security-foundations
Security-first guide for building Uniswap V4 hooks covering known vulnerability patterns, audit requirements, and Solidity best practices.

## Highlights
- Comprehensive guide to known Uniswap V4 hook vulnerability patterns and mitigations
- Audit requirements and pre-audit checklist for hook contracts
- Solidity best practices specific to the V4 hook architecture
- Coverage of hook permission flags and reentrancy risks
- Part of the uniswap-hooks plugin
- Targets Solidity developers building custom V4 pool hooks
- References official Uniswap V4 audit reports and security advisories
- Pairs with forge-based testing workflows for security validation
