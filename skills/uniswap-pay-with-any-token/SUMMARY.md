# uniswap-pay-with-any-token
Pay HTTP 402 payment challenges using any ERC-20 token via Tempo CLI and the Uniswap Trading API, supporting both MPP and x402 protocols.

## Highlights
- Pay HTTP 402 payment challenges with any ERC-20 token in the user's wallet
- Tempo CLI integration for MPP (Machine Payment Protocol) flows
- x402 protocol support for machine-to-machine payments
- Uniswap Trading API handles the token swap before payment settlement
- Works with any token on Uniswap-supported chains
- Part of the uniswap-trading plugin alongside swap-integration
- Enables AI agents to autonomously resolve payment-gated API calls
- No manual swap step required — swap and pay happen atomically
