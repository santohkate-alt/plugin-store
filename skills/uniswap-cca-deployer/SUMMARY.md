# uniswap-cca-deployer
Deploy Continuous Clearing Auction (CCA) smart contracts using the Factory pattern with CREATE2 for deterministic, consistent addresses across chains.

## Highlights
- Deploy CCA smart contracts via Factory pattern
- CREATE2 for deterministic contract addresses reproducible across chains
- Covers factory invocation, salt selection, and deployment verification
- Part of the uniswap-cca plugin suite — pairs with uniswap-cca-configurator
- Supports Ethereum and EVM-compatible chains
- Forge script integration for reproducible deployments
- AI-guided deployment flow with pre-deployment parameter review
- Post-deployment verification and address confirmation steps
